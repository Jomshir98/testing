SetHeightOverride

  • SetHeightOverride(height: null | number): void

  • Makes the bot be offset by height pixels up. The amount is limited and will be capped for other players.

    Note: this works by adding an item with manually crafted properties on bot and only works on bot itself!

    • height: null | number

      The height in pixels (allows for negative numbers); null to clear

    Returns void
