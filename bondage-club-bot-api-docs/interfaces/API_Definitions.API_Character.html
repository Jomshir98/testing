GetAllowItem

  • GetAllowItem(): Promise<boolean>

  • Queries if bot is allowed to interact with target user.

    Uses GetAllowItemQuick to return result immediately, but also does consistency check by actually asking server

    IMPORTANT: Also returns false, if bot is unable to interact with the character based on protection checks. See ProtectionAllowInteract.

    Returns Promise<boolean>

    Promise of the result

    Returns boolean
